7 """
8 BitArray{N} <: AbstractArray{Bool, N}
9
10 Space-efficient `N`-dimensional boolean array, using just one bit for each boolean value.
11
12 `BitArray`s pack up to 64 values into every 8 bytes, resulting in an 8x space efficiency
13 over `Array{Bool, N}` and allowing some operations to work on 64 values at once.
14
15 By default, Julia returns `BitArrays` from [broadcasting](@ref Broadcasting) operations
16 that generate boolean elements (including dotted-comparisons like `.==`) as well as from
17 the functions [`trues`](@ref) and [`falses`](@ref).
18 """

1200 function reverse!(B::BitVector)
1201 # Basic idea: each chunk is divided into two blocks of size k = n % 64, and
1202 # h = 64 - k. Walk from either end (with indices i and j) reversing chunks
1203 # and separately ORing their two blocks into place.
1204 #
1205 # chunk 3 chunk 2 chunk 1
1206 # ┌───────────────┬───────┐┌───────────────┬───────┐┌───────────────┬───────┐
1207 # │000000000000000│ E ││ D │ C ││ B │ A │
1208 # └───────────────┴───────┘└───────────────┴───────┘└───────────────┴───────┘
1209 # k h k h k
1210 # yielding;
1211 # ┌───────────────┬───────┐┌───────────────┬───────┐┌───────────────┬───────┐
1212 # │000000000000000│ A' ││ B' │ C' ││ D' │ E' │
1213 # └───────────────┴───────┘└───────────────┴───────┘└───────────────┴───────┘
1214
1215 n = length(B)
1216 n == 0 && return B
1217
1218 k = _mod64(n+63) + 1
1219 h = 64 - k
1220
1221 i, j = 0, length(B.chunks)
1222 u = UInt64(0)
1223 v = reverse_bits(B.chunks[j])
1224 B.chunks[j] = 0
1225 @inbounds while true
1226 i += 1
1227 if i == j
1228 break
1229 end
1230 u = reverse_bits(B.chunks[i])
1231 B.chunks[i] = 0
1232 B.chunks[j] |= u >>> h
1233 B.chunks[i] |= v >>> h
1234
1235 j -= 1
1236 if i == j
1237 break
1238 end
1239 v = reverse_bits(B.chunks[j])
1240 B.chunks[j] = 0
1241 B.chunks[i] |= v << k
1242 B.chunks[j] |= u << k
1243 end
1244
1245 if isodd(length(B.chunks))
1246 B.chunks[i] |= v >>> h
1247 else
1248 B.chunks[i] |= u << k
1249 end
1250
1251 return B
1252 end
